This beta release expands dashboard appearance options and improves configuration-service cleanup.

## New features

- Added light and dark theme support with separate background images.
- Added custom background images from local files or URLs.

## Fixes and improvements

- Added destroy subscriptions in the configuration service to prevent memory leaks.
