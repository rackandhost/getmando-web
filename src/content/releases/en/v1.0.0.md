This stable release removes beta references and adds favorites, broader test coverage, and pull-request safeguards.

## New features

- Added favorite applications, grouped in a dedicated Favorites category and available through search filtering.

## Quality and tooling

- Added component and core-service tests, including keyboard activation coverage for Space and Enter.
- Added pull-request test automation for `develop` and `main` plus CODEOWNERS approval requirements.
